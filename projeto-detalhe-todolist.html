To-Do List Interativo

Uma aplicação web simples e eficaz para gerenciamento de tarefas, construída com HTML, CSS e JavaScript puro, utilizando LocalStorage para persistência de dados.

Sobre o Projeto

O objetivo deste projeto foi desenvolver uma aplicação de lista de tarefas (To-Do List) funcional e intuitiva, utilizando tecnologias front-end fundamentais. A aplicação permite aos usuários adicionar, marcar como concluída, editar e remover tarefas. Além disso, as tarefas são salvas no LocalStorage do navegador, garantindo que os dados persistam mesmo após o fechamento da página.

Funcionalidades Implementadas

  • Adicionar Tarefas: Campo de entrada para adicionar novas tarefas à lista.
  • Marcar como Concluída: Checkbox para alternar o estado de uma tarefa entre pendente e concluída, com feedback visual.
  • Editar Tarefas: Capacidade de editar o texto de uma tarefa existente.
  • Remover Tarefas: Botão para excluir tarefas individualmente.
  • Filtrar Tarefas: Opções para visualizar todas as tarefas, apenas as ativas ou apenas as concluídas.
  • Persistência de Dados: Uso do LocalStorage para salvar e carregar as tarefas, mantendo o estado da aplicação entre sessões.
  • Design Responsivo: Interface adaptável a diferentes tamanhos de tela.

Desafios e Aprendizados

Um desafio interessante foi implementar a funcionalidade de edição "in-place" das tarefas e garantir que a atualização no LocalStorage fosse feita corretamente. Este projeto foi uma excelente oportunidade para aprofundar meus conhecimentos em manipulação do DOM com JavaScript puro, gerenciamento de estado simples e a utilização eficiente do LocalStorage. A lógica de filtragem e a atualização dinâmica da interface do usuário também foram aspectos importantes do desenvolvimento.
